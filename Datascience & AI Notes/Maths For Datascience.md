## Sets and Functions 
* A set is a well-defined collection of distinct objects, considered as an object in its own right.
* Core Definition
* A set contains elements (or members)
* Elements are distinct — no duplicates
* Elements are unordered — {a, b, c} = {c, b, a}
* Sets are well-defined — for any object, it's clear whether it belongs to the set or not
